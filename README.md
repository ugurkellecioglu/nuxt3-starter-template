# Nuxt 3 Starter Template

This repository is designed to be a quick and efficient starting point for developers who want to build web apps without having to spend a lot of time on basic configuration.


## Features
- [x] Tailwind
- [ ] Typescript
- [x] Pinia
- [x] Color Mode
- [x] I18n
- [x] Nuxt Auth
- [x] VueUse
- [x] Nuxt Icons
